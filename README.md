# Zitierstil für Zotero
Dieser Zitierstil entspricht den Vorgaben der DAF und DAZ Professuren am Institut für Germanistik der Justus-Liebig Universität Gießen und wurde für das Seminar Textkompetenzen in DAZ-Lerner:innen (WiSe 25/26) entwickelt.

Die Handreichung ist unter dem folgenden Link zu finden:
https://www.uni-giessen.de/de/fbz/fb05/germanistik/iprof/daf/dateien/handreichung_wiss_schreiben

Der Stil orientiert sich an den Richtlinien von 'Deutsche Sprache: Zeitschrift für Theorie Praxis Dokumentation', hrsg. vom IDS Mannheim.
